# Visual Attributes
The i2b2 Ontology `c_visualAttributes` column has three characters:

**Position 1:**

| Value | Concept/Modifier | Draggable | Leaf | Approximate |
| ----- | :-----: | :-----: | :-----: | :-----: |
| C |  C | N | N | | 
| D |  M | Y | N | |
| F |  C | Y | N | |
| L |  C | Y | Y | N |
| M |  C | Y | Y | Y |
| O |  M | N | N | |
| R |  M | Y | Y | | 

**Position 2:**

| Value | Meaning | Notes |
|  --- | --- | --- |
| A | Active | |
| H | Hidden | |
| I | Inactive | This is used by publishers to add version identifiers (CI) |

**Position 3:**

| Value | Meaning |
|  --- | --- |
| E | Editable |
|   | Not editable |
